# KirbyShop 

### 功能

(1)店鋪開張  
必須先開張才能購買卡比！

(2)目前的購買清單  


(3)當次的加購清單  
●如果三種類型的卡⽐各買了 a,b,c 隻，可以把他們打包成好幾個箱⼦，每⼀個箱⼦裡⾯的內容物必須⼀樣 ，且所有卡⽐都必須被裝進其中⼀個箱⼦裡。  
●促銷活動：如果總共裝了 T 箱，總價格可以再減少 2T。  

(4)離開商店
